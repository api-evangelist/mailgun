# MailgunThis is a repo for managing machine readable artifacts for the [Mailgun APIs](http://mailgun.net). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Mailgun APIs](http://mailgun.net).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://mailgun.net)- [Developer](http://mailgun.net)- [Twitter](https://twitter.com/#!/mail_gun)- [GitHub](https://github.com/mailgun)The [APIs.json](https://github.com/api-evangelist/mailgun/blob/master/apis.json) index provides links to all the relevant links for the Mailgun APIs, including the OpenAPI and Postman if they are present.
